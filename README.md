# OEngine
A simple 2D ECS game engine developed using C++ and SDL.

# Features
* Entity-Component-System (ECS) with Sparse Sets
* Object Pooling
* Uniform Grid Collision Detection
* Elastic Collision Responese
* 2D Rendering and Input Handling via SDL
